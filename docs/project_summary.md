# Credit Risk Scoring System

## Objective

The objective of this project is to predict applicant credit risk using machine learning techniques and financial information from the German Credit Dataset.

## Dataset

* Dataset: German Credit Dataset
* Records: 1000 applicants
* Features: 20
* Target Variable: Credit Risk

## Methodology

1. Data Loading and Cleaning
2. Exploratory Data Analysis (EDA)
3. Train-Test Split
4. Logistic Regression Model
5. Random Forest Model
6. Model Comparison
7. Feature Importance Analysis

## Results

| Model               | Accuracy |
| ------------------- | -------- |
| Logistic Regression | 75.0%    |
| Random Forest       | 73.5%    |

Logistic Regression achieved slightly better performance and was selected as the preferred model.

## Key Findings

* Credit Amount was the most influential feature.
* Checking Account Status strongly influenced risk prediction.
* Credit History and Loan Duration contributed significantly to model decisions.
* Multiple financial factors were required to accurately assess credit risk.

## Conclusion

This project demonstrates how machine learning can support credit risk assessment and lending decisions in financial institutions.
